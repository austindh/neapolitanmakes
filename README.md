# neapolitanmakes
[neapolitanmakes.com](https://neapolitanmakes.com)

## Development
#### Run admin server:
```bash
# from /
$ ts-node app
```

#### Run admin frontend:
```bash
# from /admin
$ npm run start
```

## Building
#### Build typescript files to be included in blog
```bash
# from /admin
$ npm run build:ts
```

#### Build site CSS
```bash
$ npm run build:css
```

#### Build all pages for the site (and CSS)
```bash
$ npm run build:blog
```
